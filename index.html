HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta tag is for the search browswer and includes things like keyworks for SEO.
    • The head element contains metadata that helps the browser render the page correctly and is not visisble when a user looks at a webpage.
  • The script element embeds the type of sclipt that will be used, for example javascript.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding element indicates how much space we want around the content inside an element.
  • '*' a wildcard symbol applies to ALL elements.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git branch : will list local branch types.
  • It's possible to push and commit directly in vs code.
  • You 'stage' changes because you might not want to commit all of them.
  • GitFlow: make sure your on right feature branch, stage changes, commit changes, pull main branch, push changes.

JavaScript

  • A function is a block of code used to execute a task
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array holds lists of data.
